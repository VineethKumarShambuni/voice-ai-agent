# Voice AI Agent 🎙️

A simple Voice AI Agent built using Python that converts speech to text, processes it with AI, and returns a voice response.

## Features
- Speech to Text
- AI Text Processing
- Text to Speech
- FastAPI based backend

## Tech Stack
- Python
- FastAPI
- Speech Recognition
- Text-to-Speech

## Run Locally

Install dependencies:

pip install -r requirements.txt

Run the server:

uvicorn main:app --reload
